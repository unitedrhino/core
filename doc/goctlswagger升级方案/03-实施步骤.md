# 03 - 实施步骤

## Step 1: 添加 kin-openapi 依赖

```shell
cd D:\code\company\go-zero\tools\goctl
go get github.com/getkin/kin-openapi@latest
```

实际安装版本：v0.133.0

## Step 2: 新建转换函数

**文件**: `tools/goctl/api/swagger/convert.go`

核心函数 `swagger2ToOpenAPI3()` 接收 Swagger 2.0 JSON 字节，返回 OpenAPI 3.0 JSON 字节。
内部调用 `openapi2conv.ToV3()` 完成转换，并对结果做可选验证。

## Step 3: 修改旧版命令 cmd.go

**文件**: `tools/goctl/api/swagger/cmd.go`

改动点：

1. import 中添加 `"fmt"` 和 `"os"`
2. `SwaggerCommand()` 末尾：先调用 `generate.Do()` 生成 Swagger 2.0 文件，再读取该文件，调用 `swagger2ToOpenAPI3()` 转换，最后覆盖写入

## Step 4: 修改新版命令 command.go

**文件**: `tools/goctl/api/swagger/command.go`

改动点：
在 `json.MarshalIndent()` 之后、写文件之前，插入 `swagger2ToOpenAPI3()` 调用，将内存中的 Swagger 2.0 数据转换为 OpenAPI 3.0

## Step 5: 编译验证

```shell
cd D:\code\company\go-zero\tools\goctl
go build -o goctl.exe .
go test ./api/swagger/... -v -run TestSwagger2ToOpenAPI3
```

单元测试和编译均已通过验证。
